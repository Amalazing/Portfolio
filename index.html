Messaging Service

A full stack web service that sends users messages every day via email. The project was built using the MERN stack and is complete with payment processing via Stripe. I utilize the Mongo database to store and access stored user information. The Node REST webserver schedules emails to be sent and handles user information sent from the front end by validating it. I planned, developed, produced, and am now maintaining this project. I am very proud of it. Feel free to check out the site itself by using the link below, or watch the video to see it in action.
