Node-CRUD-API

An API I created using Node that performs the basic CRUD operations on a hard-coded array of user objects. I implemented a web server for greater usability in terms of performing the operations and seeing the results of the operations. To use for yourself please download and extract the zip off of Github. Then install dependencies "npm i". Then run "node index". Now you can visit local-host at port 5000 for adding members, or you can use Postman to test all CRUD operations.
